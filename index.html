Paper Introduction

This project is based on our research paper, which explores advanced techniques in text-to-speech (TTS) synthesis. The proposed method leverages a generative model to achieve high-quality, natural-sounding speech. Key highlights of the paper include:

  • Eliminating the need for explicit text-to-speech alignment.
  • Adopting a two-stage model: semantic token prediction and acoustic token generation.
  • Using a large-scale dataset for training to improve zero-shot performance.
